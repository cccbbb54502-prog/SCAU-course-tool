# SCAU Course Tool 使用说明

华南农业大学教务系统抢课工具。

## 概览：
本工具重新重构了整个抢课脚本 与上一个版本完全不同的是获得了教务系统的大部分api接口，故不用通过启动浏览器来进行加密登录，相关api接口文档在docs文件夹中
## 使用前必须准备的：
需提前把要抢的课程加入教务系统的意向购物车。本工具只会读取意向购物车中的课程，并从购物车持续提交选课。

## 满课处理（捡漏功能）

当某门课返回“人数已满”等提示时，程序只会暂停或继续处理当前这门课，不会自动暂停所有课程。

如果选择继续蹲守（看看能不能等到某个人退课然后捡漏），该课程会继续提交；如果选择暂停，则只停止当前课程，其他仍在激活状态的课程会继续运行。

## 注意事项

- 请先在教务系统网页中把目标课程加入意向购物车。
- 不建议把请求间隔设置得过低，避免触发系统限制或造成账号风险。
- `config.json` 中可修改相关配置
- 如果触发短信验证或身份信息确认，当前工具不会自动完成该分支。


## 使用 exe 运行
可运行dist文件夹中的exe文件即可启动脚本工具

## 配置文件

exe 默认读取同目录下的 `config.json`：

```text
dist\config.json
```

源码运行时默认读取脚本同目录下的 `config.json`：

```text
config.json
```

配置示例：

```json
{
  "duration_seconds": 600,
  "speed_profile": "medium",
  "speed_profiles": {
    "ultra_fast": 0.01,
    "fast": 0.1,
    "medium": 0.2,
    "slow": 0.5
  }
}
```

字段说明：

| 字段 | 说明 |
| --- | --- |
| `duration_seconds` | 抢课持续时间，单位秒 |
| `speed_profile` | 当前使用的速度档位 |
| `speed_profiles` | 各速度档位对应的请求间隔，单位秒 |

速度档位示例：

| 档位 | 请求间隔 |
| --- | --- |
| `ultra_fast` | 0.01 秒 |
| `fast` | 0.1 秒 |
| `medium` | 0.2 秒 |
| `slow` | 0.5 秒 |
